開放對象

  1. 本方案服務開放申請對象，以本校在職教職員工為限，不含校內合作之廠商及校外人士、單位。
  2. 本服務係指使用者依照巨量運算與機器學習等需求，向本校電算中心提出校園AI機房虛擬主機租用申請，以取得建立其需要之人工智慧運算資源之服務模式。

 

服務內容

（一）研究方案

  1. 提供Amazon Web Services公有雲服務平台存取權限。
  2. 原則上所有等級的硬體資源都可使用，為避免操作不慎導致浪費，請申請人依研究需求告知可用軟硬體項目，由電算中心協助調整並管控使用資源。
  3. 目前可用額度約20萬元，同時間最多開放5人次申請操作，推廣試行期間採登記優先排隊制，每次最長租期為一個月，租期屆滿即中止存取權限或重新排隊申請。

（二）教學方案

  1. 提供Amazon Web Services Educate公有雲服務平台，本校在職教職員工及在學學生皆可自行上網申請。
  2. 教師通過審核後，將具有建立線上課程的權限，每次開課可給予最高50美金的免費使用額度。
  3. 免費點數用罄，虛擬機將自動停機，不會收取額外費用。
  4. 一旦點數用罄，虛擬機將無法開機，請提前備份資料，避免資料遺失。
  5. 可使用免費服務項目請參閱：https://s3.amazonaws.com/awseducate-starter-account-services/AWS_Educate_Starter_Accounts_and_AWS_Services.pdf
  6. AWS Educate免費帳號分兩種，AWS Educate Educator/Student (綁定信用卡)、AWS Educate starter Educator/Student (無綁定信用卡)，其中差異在於綁定信用卡可獲得較高的免費存取額度，學生75USD~100USD、教職員工100USD~200USD。簡易流程如下：
    • 至https://aws.amazon.com/tw/education/awseducate/填寫申請表格，e-mail請使用本校webmail申請
    • 如需使用綁定信用卡帳戶請先至https://aws.amazon.com/tw/free/申請AWS一般免費帳號；再至步驟6.1網址申請AWS Educate免費帳號。
    • AWS Educate免費帳號申請完畢後，請等待10天左右，待審核完畢後，依「核可信」指示即可完成AWS Educate免費帳號申請建置。

申請人同意：

  1. 遵守任何適用之國家制訂法律規章、約定條款(包括「國立臺中科技大學校園網路使用規範」、教育部「台灣學術網路使用規範」等)以及良好的禮儀，尊重他人隱私。
  2. 本服務以支援本校教學、學術研究活動、公務行政為主要目的，因系統資源有限，嚴格禁止超出申請用途範圍以外行為。
  3. 使用者違反前述相關規定時，本校電算中心保有隨時中斷其使用之權力。
  4. 推廣試用期間，請配合電算中心調查使用情況，如課程名稱、內容、上課學生人數、選用雲端資源、服務、意見回饋等，俾利蒐集推廣資料，或依需求調整後臺管理模式。
  5. 請尊重智慧財產權，嚴禁下載、安裝不法軟體，嚴禁用於干擾破壞或嘗試侵入未經授權之電腦系統，如散佈電腦病毒、蠕蟲、後門、木馬等其它類似之情況，違法者自負相關法律責任。
  6. 公有雲服務目前機房皆在海外，如有機敏性資料，請自行參照國內法規判斷，遵守法律規章。
  7. 請自行管控使用情況，公有雲虛擬機一旦啟動，即開始計算費用，即使停止亦會收取儲存體的費用，若使用結束務必停止或刪除，避免造成浪費。
