緣起

電算中心配合本校辦理國立技專校院校務發展特色躍升計畫，以推廣全校師生學習人工智慧運算暨雲端服務應用技術，本中心將提供本校校內人工智慧運算暨雲端服務，並協助全校師生將人工智慧運算研究發展或教學負載移至雲端服務。

 

服務簡介

  • AI+VDI 虛擬桌面：建置於三民校區資訊館2F機房，採用VMware Horizon虛擬化桌面技術，搭配Nvidia Tesla V100的GPU運算資源，提供人工智慧的運算環境。
  • Amazon Web Services：目前全球公有雲服務的市佔率最高的廠商，隨選隨用，以量計價的營運模式，全球超過70個可用區域，提供多樣化的雲端服務。本校亦參加Amazon雲端服務教育方案，師生以學校信箱(ePortal@nutc.edu.tw)註冊，可獲得每年更新的免費額度點數及部份服務。
  • Microsoft Azure：全球公有雲服務的市佔率第二的廠商，同樣採用隨選隨用，以量計價的營運模式，全球超過50座資料中心。微軟亦提供教育方案，師生以學校帳號(ePortal@ad1.nutc.edu.tw)註冊，可獲得一次性的免費額度。

 

開放對象

  • 本方案服務開放申請對象，以本校在職教職員工為限，不含校內合作之廠商及校外人士、單位。
  • 本服務係指使用者依照巨量運算與機器學習等需求，向本校電算中心提出校園AI機房虛擬主機租用申請，以取得建立其需要之人工智慧運算資源之服務模式。

 

優點

  • AI+VDI建置於校內機房，操作介面簡易，已安裝GPU運算基礎套件，適合大量進行人工智慧運算，或校內教學人工智慧課程使用。
  • Amazon Web Services 及 Microsoft Azure 適合快速學習及開發各種新型態的應用服務(大數據、串流服務、語音助理、預測分析、人工智慧、IoT等)，節省建置及維護基礎設施的人力成本。

 

其它資訊

推廣試行期間資源有限，服務採登記優先制，隨到隨審，服務內容詳情請洽電算中心，歡迎踴躍申請，謝謝。

欲申請本服務的教職員，請下載AI機房申請表單，依需求填寫完成後，將正本送至電算中心辦理。

未來各項服務將蒐集使用者體驗回饋，並統整各項因素評估後，將逐步進行調整作業，以期達到推廣試行的最佳效益。
