開放對象

  1. 本方案服務開放申請對象，以本校在職教職員工為限，不含校內合作之廠商及校外人士、單位。
  2. 本服務係指使用者依照巨量運算與機器學習等需求，向本校電算中心提出校園AI機房虛擬主機租用申請，以取得建立其需要之人工智慧運算資源之服務模式。

 

服務內容

（一）研究方案

  1. 提供Microsoft Azure公有雲服務平台存取權限。
  2. 原則上所有等級的硬體資源都可使用，為避免操作不慎導致浪費，請申請人依研究需求告知可用軟硬體項目，由電算中心協助調整並管控使用資源。
  3. 目前可用額度約6萬元，同時間最多開放3人次申請操作，推廣試行期間採登記優先排隊制，每次最長租期為一個月，租期屆滿即中止存取權限或重新排隊申請。
  4. 公有雲服務目前機房皆在海外，如有機敏性資料，請自行參照國內法規判斷，遵守法律規章。

（二）教學方案

  1. 提供Microsoft Azure公有雲服務平台存取權限。
  2. 原則上所有等級的硬體資源都可使用，為避免操作不慎導致浪費，請申請人依研究需求告知可用軟硬體項目，由電算中心協助調整並管控使用資源。
  3. 本方案申請依據以本校每學期開課為基準，預計可開5門課，每門課可用額度約新臺幣3萬元，額度用罄即無法使用。
  4. 一旦額度用罄，虛擬機將無法開機，請提前備份資料，避免資料遺失。

 

使用規範

申請人同意：

  1. 遵守任何適用之國家制訂法律規章、約定條款(包括「國立臺中科技大學校園網路使用規範」、教育部「台灣學術網路使用規範」等)以及良好的禮儀，尊重他人隱私。
  2. 本服務以支援本校教學、學術研究活動、公務行政為主要目的，因系統資源有限，嚴格禁止超出申請用途範圍以外行為。
  3. 使用者違反前述相關規定時，本校電算中心保有隨時中斷其使用之權力。
  4. 推廣試用期間，請配合電算中心調查使用情況，如課程名稱、內容、上課學生人數、選用雲端資源、服務、意見回饋等，俾利蒐集推廣資料。
  5. 請尊重智慧財產權，嚴禁下載、安裝不法軟體，嚴禁用於干擾破壞或嘗試侵入未經授權之電腦系統，如散佈電腦病毒、蠕蟲、後門、木馬等其它類似之情況，違法者自負相關法律責任。
  6. 公有雲服務目前機房皆在海外，如有機敏性資料，請自行參照國內法規判斷，遵守法律規章。
  7. 請自行管控使用情況，公有雲虛擬機一旦開啟，即開始計算費用，即使停止亦會收取儲存體的費用，若使用結束務必停止或刪除，避免造成浪費。

 

隱私聲明

本校因申請人使用本服務而蒐集之個人資訊將受到本校《隱私權政策》之保護。如 申請人註冊本服務時即表示已理解並完全同意接受。

 

免責聲明

  1. 請用戶定期備份資料，電子計算機中心不負資料保管之責任，若有資料遺失損毀之情形，恕難提供協助。服務一旦中止，資料即無法取得，請事前備份資料。
  2. 公有雲端服務資源為供應商所提供租用，服務可用性需考量供應商實際營運狀況，由於本校沒有實體設備的管理權，將無法保證服務的穩定度。
